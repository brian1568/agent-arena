# Agent Arena 🤖⚡

Welcome to **Agent Arena**. This repository serves as a centralized testing and evaluation ground for multiple AI coding assistants, agents, and programming tools. 

## Structure

The repository organizes projects **by the AI Agent** that authored them. This allows isolated experimentation and side-by-side benchmarking of how different tools tackle architectural decisions, file scaffolding, and logical components.

```text
agent-arena/
├── antigravity/
│   ├── classic-snake/     # Vanilla JavaScript Snake game implementation
│   └── ... 
├── generic-copilot/
│   └── ...
└── README.md
```

## Getting Started

To explore a specific agent's work, navigate into their dedicated directory (e.g., `cd antigravity/classic-snake`). Each individual project contains its own `README.md` outlining specific run instructions, dependencies, and test commands.
